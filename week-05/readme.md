# Domain Name vs FQDN vs URL

- Domain Name:網域名稱
- FQDN:完整的網域名稱
- URL:具體的位址

完整性:URL>FQDN>Domain Name

# 為什麼應該要為網站加上憑證?

1. 加上憑證後，網站的資料就會加密
2. 提高網站的可信度
3. 如果加密資料被破解可求償
